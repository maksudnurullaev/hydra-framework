What is Hydra?

Apr26, 2010 - Hydra, yet another web applicaton framework.

The existence of huge open source tools for web development is sometimes terrified, instead of make a right choise to solve the problem. But we believe that web-developers just needs:

   ·  flexible language (like java and javasript);
   ·  light customizable libraries (like Spring, DWR and Mootools);
   ·  and scalable database (like Apache Cassandra).

Hydra is web application framework with combination of all described tools and libraries.
