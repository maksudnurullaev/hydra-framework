Installing A Template

To install one of these WordPress templates:

1. Download the archive and extract the folder containing all of the files.

2. Using an FTP client to access your host web server, upload the theme to the wp-content/themes directory in WordPress.

3. Select the new theme in the Apperance > Themes admin screen.

As with any WordPress template, you'll more than likely need to adjust the styles a bit to mesh with your current settings.

(NOTE: Web geek tested and approved in Firefox 3.0 (Win), Firefox 2.0 (Mac), IE7 (Win), Opera 9.6 (Win), Safari 3.2 (Mac). IE6 can go to hell.)
