История

Cуществование огромного количества свободный программ и инструментов для веб разработок иногда ужасает, а не радует. Мы полагаем что для "счастья" веб-программиста необходимы лишь три компоненты:
  • легкая, гибкая и программируемая серверная часть;
  • легкая, гибкая и программируемая клиентская часть;
  • легкая, гибкая и программируемая связь между клиентской и серверной частью.
Из этих соображений, для нашего проекта, мы выбрали такие компоненты:
  • серверная часть - The Spring Framework (открытая платформы для Java разработок);
  • клиентская часть - MooTools (свободный JavaScript фреймворк для разработки кроссбраузерных веб-приложений и веб-сервисов);
  • и то, что их свяжет - DWR (свободная Java библиотеки, упрощающая взаимодействие Java скриптов с Java сервером).
"До кучи" добавили собственные разработки по многопоточному выполнению запросов и назвали это чудо HydraFramework-ом!

Помните, что вы всегда свободны в выборе и можете использовать совершенно другие компоненты и вариации на эту же тему, но это будет уже другая история... ;-)
